# Machine Learning Trading Bot

To build a trading bot with Machine Learning.

## Technologies

Programming Language is Python
Version Control system is Github
Git Bash
Jupyter Lab



Libraries are given below
pandas
numpy
pathlib
hvplot
matplotlib
sklearn

## Installation Guide

Install below libraries
pandas
numpy
pathlib
hvplot
matplotlib
sklearn
